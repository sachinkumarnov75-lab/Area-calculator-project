# Area Calculator

A beginner friendly Python project that calculates the area of different geometric shapes through a simple menu-driven interface.

## Supported Shapes

* Triangle
* Rectangle
* Square
* Circle

## Features

* Interactive command-line menu
* User input handling
* Area calculations for multiple shapes
* Exit option for terminating the program
* Beginner-friendly Python code structure

## Technologies Used

* Python 3

## How to Run

1. Clone the repository:
git clone <<add-repository-link>>

2. Navigate to the project folder:
cd area-calculator-python

3. Run the program:
python3 area.py

## Learning Objectives

This project was built to practice:
